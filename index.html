Produk yang kami tawarkan

Manrisk

Manrisk adalah aplikasi menejemen risiko, untuk mempermudah perusahaan Anda dalaman mengetur risiko diperusahaan Anda. Dengan aplikasi ini Anda dipermudah untuk mengidentifikasikan risiko, menangani risiko dan menganalisa risiko.

Proment

Proment adalah aplikasi project management, untuk mempermudah Anda dalam merencanakan project mulai dari task, budgeting sampai realisasinya, tidak hanya itu tetapi dengan proment Anda bisa mengevaluasi dari project anda secara realtime.

CRM

CRM (Customer Relationship Management)adalah aplikasi untuk menjaga hubungan baik dengan pelanggan, dengan Apliasi CRM anda dipermudah untuk mengatur data customer, event dengan customer, info ulang tahun customer, keluhan, potensi proyek/customer dll.
